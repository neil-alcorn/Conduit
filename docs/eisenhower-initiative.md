# The Eisenhower Initiative

The Eisenhower Initiative is the structured program for bringing every Horace Mann repo onto the CONDUIT network.

## Highway Init Steps

1. Register the repo in `highway-index/`.
2. Create the repo's initial `CONDUIT.md` and `CONTEXT.md`.
3. Set Repo Signals, escalation contacts, audience defaults, and LeanIX ID.
4. Link the repo to its ADO project and acceptance criteria registry.
5. Clear `QUARANTINE` only after validation succeeds.

## Prioritization

1. Highest Field Agent and Customer impact repos.
2. Actively developed repos.
3. Maintenance-only repos.
4. Archived or inactive systems.
