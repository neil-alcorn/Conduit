# CONDUIT Glossary

- `Convoy`: the end-to-end unit of intent.
- `Work Stream`: the repo-scoped unit of execution within a convoy.
- `Checkpoint`: the atomic unit of agent work.
- `Information Highway`: the deterministic context document set an agent reads before acting.
- `Gate`: the human approval boundary between stages.
- `Gate Sync`: the pre-gate refresh that ensures decisions use current state.
- `Repo Signal`: the structured metadata block that tells agents what they may do in a repo.
- `Eisenhower Initiative`: the phased Highway Init program for bringing repos onto the CONDUIT network.
